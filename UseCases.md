# Use Cases for Capstone Pickleball Mobile

## Court Management

### Create Court

- **Actor:** Admin
- **Description:** Admin creates a new court in the system by providing required details.

### Update Court

- **Actor:** Admin
- **Description:** Admin updates the information of an existing court.

### Delete Court

- **Actor:** Admin
- **Description:** Admin removes a court from the system.

### View Court

- **Actor:** Admin
- **Description:** Admin views the list of courts or details of a specific court.
